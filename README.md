# Python Cybersecurity Projects

This repository contains Python projects completed during a 2-week UOP Summer Institute program (2024).

## Projects
- Encryption/Decryption: Programs to encode and decode messages
- Password Hashing: Scripts to analyze password strength
- Brute Force Simulation: Demonstrates password cracking techniques
- Keylogger (Educational): Demonstrates how keystroke logging works
- Basic Programs: Calculator and other Python exercises

## Skills Used
- Python fundamentals (loops, conditionals, functions)
- Basic cybersecurity concepts
- Problem-solving in a team environment
